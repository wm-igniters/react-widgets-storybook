# Styling

The Accordion component can be styled using the theme's styling options. The component is built to adapt to the application's theme automatically.

## Accordion Pane Styling

Each accordion pane can be customized with:

- Title text and formatting
- Optional subtitle text
- Custom icon via icon class
- Badge with configurable styling (primary, success, info, warning, danger)

## Badge Types

Badges can be styled with the following types:

- `default`: Standard badge styling
- `primary`: Primary theme color
- `success`: Green color indicating success
- `info`: Blue color for informational badges
- `warning`: Yellow/orange color for warnings
- `danger`: Red color for errors or important notices
