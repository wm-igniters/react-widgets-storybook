## Overview

The Marquee component is an advanced UI element that creates a smooth scrolling text or content container. It enables content to automatically scroll horizontally across the screen, similar to traditional HTML marquees but with enhanced performance and styling options. This component is ideal for displaying announcements, news tickers, or any content that needs to continuously cycle across the view.

## Features
- Seamless horizontal scrolling animation
- Customizable scroll speed and direction
- Support for both text and rich HTML content
- Pause on hover functionality
- Responsive design that adapts to container width
- Accessibility support with screen reader compatibility
