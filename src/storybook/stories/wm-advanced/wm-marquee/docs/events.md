# Callback Events

The Marquee component provides the following callback events for interaction control:

| Event | Description |
|-------|-------------|
| `onStart` | Triggered when the marquee animation begins |
| `onFinish` | Triggered when all loops are completed (only if loop is not infinite) |
| `onPause` | Triggered when animation is paused (via pauseOnHover or manually) |
| `onResume` | Triggered when animation resumes after being paused |
| `onHover` | Triggered when user hovers over the marquee |
| `onLeave` | Triggered when user's cursor leaves the marquee |
