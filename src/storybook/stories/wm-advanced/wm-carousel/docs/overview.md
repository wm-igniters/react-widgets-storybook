## Overview

The Carousel component is an advanced UI element designed for displaying a collection of content (images, cards, text) in a rotating sequence. It provides a dynamic way to showcase multiple items in a limited space, allowing users to navigate through the content either automatically or manually.

## Features:
* Customizable dimensions with width and height properties
* Built-in animation capabilities
* Responsive design adapting to various screen sizes
* Supports both automatic rotation and manual navigation
* Compatible with various content types (images, videos, HTML content)
