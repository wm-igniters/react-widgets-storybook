# Overview

The `login-dialog` component provides a modal dialog interface for user authentication within an application. This reusable component handles the user login process with a clean, customizable UI that can be triggered programmatically and responds with appropriate event callbacks based on login success or failure.

## Features
- Modal dialog presentation for username/password authentication
- Customizable dialog title and icon
- Configurable button text for login and cancel actions
- Error message display capabilities
- Rich event handling for login lifecycle (submit, success, error, open, close)
- Integration with application authentication systems
