# Overview

The `page-dialog` component is a specialized dialog widget designed to render a page as a dialog, providing a full-featured and flexible dialog experience within your application. This component allows developers to display complex content in a modal context, using either custom content or predefined prefab templates.

## Features

- Display rich content in a modal dialog format
- Option to use either custom content or prefabricated templates
- Seamless integration with existing page structures
- Consistent user experience with the application's design language
- Configurable through properties for various use cases
