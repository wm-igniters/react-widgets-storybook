# Overview

The iframe-dialog component is a specialized dialog that displays web content from an external URL within an iframe. It provides a seamless way to embed external content within your application while maintaining control over the presentation, such as headers, actions, and modal behavior. This component is particularly useful for integrating third-party content, showing documentation, or embedding other web-based resources without requiring users to leave your application.

## Features

- Embed external web content in a controlled dialog environment
- Configurable header with customizable title and icon
- Optional modal behavior to block interaction with the underlying page
- Customizable action buttons and close controls
- Support for URL encoding for properly handling complex URLs
- Responsive design that adapts to different screen sizes
- Configurable display settings for header and action areas
