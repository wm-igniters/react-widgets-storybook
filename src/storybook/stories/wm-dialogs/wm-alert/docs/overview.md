# Overview

The Alert Dialog component is a specialized modal dialog that presents important information to users, typically requiring acknowledgment. It's designed to interrupt the user's workflow to communicate critical messages, errors, warnings, or confirmations that need immediate attention.

Alert dialogs are commonly used to notify users about system events, completion of processes, errors that have occurred, or to confirm user actions before proceeding with potentially destructive operations.

## Features

- Customizable dialog title and message text
- Support for different alert types (error, warning, success, info)
- Custom icon support through icon class property
- Single confirmation button with customizable text
- Event callbacks for button click actions
- Simple implementation with minimal configuration
