# Overview

Dialog is a UI component that creates a pop-up window that appears over the main content of an application, requiring user interaction before they can return to the main interface. The Design Dialog is specifically a form-based dialog with predefined structure to collect information from users.

The display of the dialog at runtime is typically associated with events such as button clicks, or can be triggered programmatically through JavaScript methods.

## Features
- Modal and non-modal dialog options
- Configurable header with title and close button
- Customizable width and height
- Animation support
- Icon customization
- Programmatic open/close control
