# Styling

The Currency widget supports customization through various style properties. Mobile applications typically use platform-specific styling methods rather than CSS classes.

## Basic Style Properties

- **Background Color**: Sets the background color of the currency input field.
- **Border Color**: Controls the color of the border around the input field.
- **Text Color**: Defines the color of the input text.
- **Font Size**: Adjusts the size of the text in the currency field.
- **Font Weight**: Controls the weight (boldness) of the text.
- **Border Width**: Sets the width of the border around the input.
- **Border Radius**: Controls the curvature of the input field's corners.

## State-Based Styling

- **Focus State**: Apply specific styles when the widget has focus.
- **Disabled State**: Custom appearance for when the widget is disabled.
- **Error State**: Visual indicators when validation fails.

Note that specific styling implementation may vary depending on your mobile framework implementation. Consult your platform-specific documentation for detailed styling options.
