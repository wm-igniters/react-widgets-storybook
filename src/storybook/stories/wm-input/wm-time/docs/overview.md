# Overview

The Time component is a specialized input widget designed for capturing and managing time-related data in applications. It provides users with an intuitive interface to select time values through a dedicated time picker with customizable display formats.

## Features
* Customizable time pattern for display formatting
* Configurable output format for data processing
* Adjustable hour and minute step increments
* Support for both 12-hour and 24-hour time formats
* Time picker visibility control (on input click or button click only)
* Accessibility features including tab indexing and shortcut keys
* Support for validation rules including required field validation
