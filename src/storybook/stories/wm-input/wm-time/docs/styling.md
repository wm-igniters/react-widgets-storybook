# Styling

The Time component in mobile applications can be styled using the provided styling properties and custom styles.

## Basic Styling

The Time component can be styled using the following approaches:

1. **Theme Customization**: Apply styling through the app's theme settings
2. **Component-Specific Styling**: Apply styles directly to the component

## Styling Properties

| Class/Property | Description |
| --- | --- |
| width | Controls the width of the time component |
| timepicker-container | Affects the overall time picker container |
| time-input | Styles the input field portion of the time component |
| time-button | Affects the button that opens the time picker |

## Layout Styling

The time component's dimensions and layout can be controlled using standard layout properties:

- **Width**: Can be specified in px or % (e.g., 50px, 75%)
- **Margin & Padding**: Can be applied to position the component appropriately within its container

## Time Picker Styling

The time picker popup appearance can be customized through theme customization. This includes:

- Time picker background color
- Selected time highlighting
- Hour/minute selector appearance
- AM/PM selector styling (when using 12-hour format)
