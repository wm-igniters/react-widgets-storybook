# Callback Events

| Event | Description |
| --- | --- |
| onChange | Triggered when the time value changes |
| onFocus | Triggered when the component receives focus |
| onBlur | Triggered when focus leaves the component |

## Touch Events
| Event | Description |
| --- | --- |
| onTap | Triggered when the component is tapped |

## Mouse Events
| Event | Description |
| --- | --- |
| onClick | Triggered when the component is clicked |
| onMouseEnter | Triggered when the mouse pointer enters the component area |
| onMouseLeave | Triggered when the mouse pointer leaves the component area |
