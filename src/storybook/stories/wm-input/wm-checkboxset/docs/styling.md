# Styling

The CheckboxSet component's appearance can be customized through appropriate styling properties. The following are key style aspects that can be modified:

| Style Aspect | Description |
|-------------|-------------|
| Container Styling | Affects the overall container of the checkbox set |
| Checkbox Styling | Modifies the appearance of individual checkboxes |
| Label Styling | Changes the styling of checkbox labels |
| Group Header Styling | Customizes the appearance of group headers when grouping is enabled |

## Core Style Classes

- `.checkbox-set-container` - The main container for the checkbox set
- `.checkbox-item` - Individual checkbox items within the set
- `.checkbox-label` - Labels for each checkbox option
- `.checkbox-group-header` - Headers for grouped checkboxes when using groupBy
- `.checkbox-group-count` - Element displaying the count when showCount is enabled

Note that specific styling implementations may vary based on the React Native theme being used in the application.
