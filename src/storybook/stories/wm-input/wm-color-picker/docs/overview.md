# Overview

The color-picker component is an interactive input element that enables users to select colors from a visual palette. It provides a user-friendly interface for choosing colors in various formats (such as HEX, RGB, or HSL) without requiring manual color code entry.

## Features
- Visual color selection from a palette
- Support for multiple color formats (HEX, RGB, HSL)
- Customizable default color
- Ability to display recently used colors
- Accessible design with keyboard navigation
- Integration with form submission workflows
