# Overview

A Radioset component in React allows you to group a set of radio buttons under a common heading. It provides users with a way to select a single option from multiple choices. Radioset is particularly useful when you need to present exclusive options where only one selection is allowed at a time.

## Features

- Group multiple radio buttons under a single component
- Supports horizontal and vertical layouts
- Configurable display of items per row for different device sizes
- Data binding capabilities to dynamically populate options
- Customizable data display through expression support
- Accessibility support with configurable tab indexing
