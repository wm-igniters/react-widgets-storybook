# Overview

The Rating component allows users to display or collect feedback through a visual star rating system. It can be used for showing product ratings, collecting user feedback, or any other scenario where a quantitative or qualitative rating is needed.

## Features

- Customizable icon color through styling properties
- Optional captions that can be shown or hidden
- Configurable maximum value (number of stars) up to 10
- Flexible dataset binding for mapping captions to rating values
- Support for read-only mode for display purposes
- Keyboard navigation with arrow keys when focused
- Interactive hover effects with caption changes
