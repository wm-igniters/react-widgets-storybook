# Overview

The Select component provides users with a dropdown list of choices, allowing them to select one or more options from a predefined list. It's an essential input component for collecting user choices in a space-efficient way.

## Features

- Single or multiple selection modes
- Customizable placeholder text
- Support for large datasets
- Accessibility-friendly with keyboard navigation
- Bindable to variables for dynamic option population
- Customizable display fields and values
- Advanced display expression support for formatted options
