# Styling

The Select component can be styled using various properties and custom classes. Here are the key styling options:

## Layout Properties

| Property | Description |
|----------|-------------|
| width | Sets the width of the component in px or % (e.g., "100px" or "50%"). |
| height | Sets the height of the component in px or % (e.g., "30px" or "100%"). |

## Custom Styling

You can apply custom styling to the Select component by creating and applying your own styles in the app's stylesheet:

- Custom background colors
- Border styling
- Text formatting for options and placeholder
- Dropdown menu appearance
- Focus and hover state styling

### Common Styling Examples

```javascript
// Set width and height
Page.Widgets.mySelect.width = "250px";
Page.Widgets.mySelect.height = "40px";
```
