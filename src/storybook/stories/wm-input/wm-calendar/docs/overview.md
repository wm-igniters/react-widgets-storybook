# Overview

The Calendar component is a flexible scheduling and event display widget for React applications. It allows developers to display and manage events in various calendar views, providing intuitive date navigation and event visualization capabilities.

## Features

- Display events with customizable start and end times
- Three view options: Month, Week, and Day views
- Event selection with configurable selection modes
- Navigate between months, weeks and days
- Event customization with custom styling via class names
- Support for all-day events
- Event interaction capabilities (click, resize, drop)
