# Callback Events

| Event | Description |
| --- | --- |
| onSelect | Triggered when a user selects one or more dates |
| onViewRender | Triggered when the calendar view is changed |
| onEventDrop | Triggered when dragging of an event stops and the event has moved to a different day/time |
| onEventResize | Triggered when resizing of an event stops and the event has changed in duration |
| onEventClick | Triggered when an event is clicked in the calendar |
| onEventRender | Triggered when an event is rendered |
