# Styling

The Icon component can be styled using custom CSS classes. You can set specific styling properties like color, background, padding, etc. through your application's stylesheet.

For icon appearance and behavior, you should primarily use the component properties like `iconsize` and `iconposition` rather than custom CSS when possible.

Icon libraries like Font Awesome provide their own CSS classes for additional styling options such as:

- Size modifiers (e.g., `fa-lg`, `fa-2x`, `fa-3x`)
- Animation classes (e.g., `fa-spin`, `fa-pulse`)
- Orientation classes (e.g., `fa-rotate-90`, `fa-flip-horizontal`)
