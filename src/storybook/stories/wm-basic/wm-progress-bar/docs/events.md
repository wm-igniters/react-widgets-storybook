# Callback Events

## Mouse Events

| Event | Description |
|-------|-------------|
| onClick | Triggered when the user clicks on the widget |
| onDblclick | Triggered when the user double clicks on the widget |
| onMouseenter | Triggered when the mouse pointer enters the widget |
| onMouseleave | Triggered when the mouse pointer leaves the widget |

## Touch Events

| Event | Description |
|-------|-------------|
| onTap | Triggered when the user taps on the widget |
| onDoubletap | Triggered when the user double taps on the widget |
| onLongtap | Triggered when the user performs a long tap on the widget (React Native only) |

## Progress Events

| Event | Description |
|-------|-------------|
| onStart | Triggered when the progress begins |
| onComplete | Triggered when the progress completes (reaches 100% or maximum value) |
| onBeforeupdate | Triggered before the progress value is updated |
| onTooltiptext | Used to get the custom tooltip text for the progress bar |
