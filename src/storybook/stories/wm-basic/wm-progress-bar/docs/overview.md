# Overview

The Progress Bar widget visually represents the completion status of a task or process. It displays progress as a horizontal bar that fills from left to right as the task advances, providing users with a clear indication of how much of the task has been completed.

## Features

- Displays progress as a percentage or fraction of work completed
- Multiple visual styles including default, striped, success, info, warning, and danger
- Configurable minimum and maximum values
- Optional tooltip display
- Support for indeterminate progress states
- Customizable caption placement
- Configurable display format for progress values
