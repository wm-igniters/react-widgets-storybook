# Styling

The Progress Bar widget can be styled using the following classes:

| Class | Description |
|-------|-------------|
| .app-progress-bar | Applied to the outer container of the progress bar |
| .progress-bar-inner | Applied to the inner bar that fills to show progress |
| .progress-caption | Applied to the caption text that shows the progress value |

The Progress Bar can be styled using different visual types:
- default: Standard progress bar with blue color
- default-striped: Standard progress bar with blue striped pattern
- success: Green progress bar for successful operations
- success-striped: Green striped progress bar
- info: Light blue progress bar for informational status
- info-striped: Light blue striped progress bar
- warning: Yellow progress bar for warnings
- warning-striped: Yellow striped progress bar
- danger: Red progress bar for errors or critical states
- danger-striped: Red striped progress bar
