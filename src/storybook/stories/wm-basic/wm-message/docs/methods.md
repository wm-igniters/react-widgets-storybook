# Methods

<details open>
  <summary>Methods</summary>
    <div>
        | Method | Parameters | Return Type | Description |
        |--------|------------|------------|-------------|
        | `showMessage` | None | void | Displays the message component |
        | `hideMessage` | None | void | Hides the message component |
    </div>
</details>

### Use Cases

- Display a message.

```javascript
    Page.Widgets.message.showMessage();
```

- Hide the currently displayed message.

```javascript
    Page.Widgets.message.hideMessage();
```
