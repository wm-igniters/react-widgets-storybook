# Callback Events

| Event | Description |
|-------|-------------|
| **Mouse Events** |
| onClick | Triggered when the picture is clicked. |
| onDoubleClick | Triggered when the picture is double-clicked. |
| onMouseEnter | Triggered when the mouse pointer enters the picture area. |
| onMouseLeave | Triggered when the mouse pointer leaves the picture area. |
| **Touch Events** |
| onTap | Triggered when the picture is tapped on a touch device. |
| onDoubleTap | Triggered when the picture is double-tapped on a touch device. |
| onLongTap | Triggered when the user performs a long tap on the picture. Only supported in React Native applications. |
