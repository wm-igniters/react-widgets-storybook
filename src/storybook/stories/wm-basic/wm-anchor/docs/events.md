# Callback Events

## Focus Events

| Event | Description |
| --- | --- |
| `onFocus` | Triggered each time the anchor receives focus. |
| `onBlur` | Triggered each time focus leaves the anchor. |

## Mouse/Touch Events

| Event | Description |
| --- | --- |
| `onClick` | Triggered when the anchor is clicked/tapped. |
| `onDoubleTap` | Triggered when the anchor is double tapped. |
| `onLongTap` | Triggered when the anchor is tapped and held for a longer duration. |

## Touch-Specific Events

| Event | Description |
| --- | --- |
| `onTap` | Triggered whenever the anchor is tapped. |
| `onDoubleTap` | Triggered whenever the anchor is double tapped. |
| `onLongTap` | Triggered whenever the long tap event occurs on the anchor. |
