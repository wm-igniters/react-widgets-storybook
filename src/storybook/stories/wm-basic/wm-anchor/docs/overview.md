# Overview

The **Anchor** component in React serves as a clickable text link to navigate between screens or trigger actions. It provides a simple and intuitive way to create hyperlinks within your application.

## Features

- **Clickable Text Links**: Create clickable text links with customizable captions
- **Icon Support**: Add icons with configurable position, size, and margin
- **Badge Integration**: Display badge values for notification-style indicators
- **Navigation**: Configure links to navigate within the app or to external URLs
- **Accessibility**: Support for hints and tab indexing for better accessibility
- **Customizable Appearance**: Control text alignment and styling
