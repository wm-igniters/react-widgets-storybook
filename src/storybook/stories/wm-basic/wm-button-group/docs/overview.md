# Overview

Button Group is a widget that organizes a set of buttons that can be managed and styled as a group. It provides a structured way to display related actions together, enhancing the user interface's clarity and aesthetics.

## Features

- Groups multiple buttons together as a cohesive unit
- Supports both horizontal and vertical alignment of buttons
- Maintains consistent styling across grouped buttons
- Perfect for displaying related actions (like media controls: play, pause, stop)
- Integrates seamlessly with the grid system
- Improves visual organization of multiple related actions
- Simplifies management of button collections
