# Overview

The `Nav` component is a versatile navigation element used to create various types of navigational structures within an application. It provides a structured way to organize links, menus, and navigation hierarchies with support for different layouts, icons, badges, and nested navigation items.

This component can be configured to function as a traditional navbar, sidebar navigation, tab navigation, or a dropdown menu system, making it highly adaptable to different UI requirements.

## Features

- Multiple layout options including horizontal and vertical orientations
- Support for different navigation types (navbar, sidebar, tabs, etc.)
- Automatic opening and closing behaviors
- Icon integration with customizable positioning
- Hierarchical navigation with nested children
- Badge support for notification indicators
- Role-based visibility control
- Hover-activated navigation options
- Custom styling through class properties
