# Overview

The Table component is a versatile data display widget that enables users to organize, view, and interact with tabular data. It provides extensive functionality for data manipulation including sorting, filtering, pagination, row selection, and in-line editing. The component is designed to handle large datasets efficiently with options for on-demand loading and customizable display settings.

## Features

- **Data Management**: Display and manage tabular data with support for large datasets
- **Editing Capabilities**: In-line table editing with configurable edit modes
- **Row Manipulation**: Add, update, and delete rows with confirmation options
- **Selection**: Single (radio) or multi-select functionality
- **Sorting & Filtering**: Column-based sorting and filtering capabilities
- **Pagination**: Customizable pagination with navigation controls
- **Row Expansion**: Expandable rows for displaying additional details
- **Export**: Data export in various formats with customizable options
- **Responsive Design**: Adjustable spacing and layout options
- **Customizable UI**: Configurable headers, row indices, and visual indicators
