# Overview

The Card component is a versatile UI element designed to display content in a visually contained and organized manner. It serves as a container for related information and can include various elements such as titles, subheadings, icons, images, and action buttons. The component is highly customizable and can be used in a wide range of applications from displaying user profiles to product information.

## Features

- Customizable header with title and subheading
- Support for icons via class names or direct URLs
- Image display capabilities with configurable height
- Configurable action buttons
- Interactive behaviors with various mouse events
- Animation support
- Responsive sizing with configurable width and height
- Support for nested content through children props
