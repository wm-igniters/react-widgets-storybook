# Overview

The List component is a versatile data container that displays a collection of items in a structured format. It provides extensive customization options for data presentation, including pagination, sorting, grouping, and multi-selection capabilities. The List component can work with static datasets or dynamically fetch data through a live variable configuration.

## Features
- Flexible data binding with static arrays or live data sources
- Customizable item rendering with support for custom templates
- Built-in pagination with configurable page sizes
- Sorting and grouping capabilities
- Multi-select functionality with selection limits
- Drag-and-drop reordering
- Responsive layout with configurable items per row
- Directional support (horizontal/vertical layouts)
- Custom styling options for list and individual items
